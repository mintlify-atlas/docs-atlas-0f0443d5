## Overview

Prompt templates allow you to create reusable prompts with variable substitution. LangChain provides templates for both string-based prompts and chat-based prompts.


Create a prompt template from a template string.

<ParamField path="template" type="str" required>
  Template string with variables in `{variable}` format
</ParamField>

<ParamField path="template_format" type="str" default="'f-string'">
  Template format (`'f-string'`, `'jinja2'`, or `'mustache'`)
</ParamField>

<ParamField path="partial_variables" type="dict | None">
  Variables to pre-fill

### Example

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template(
    "Tell me a joke about {topic}"
)
prompt.invoke({"topic": "cats"})  # PromptValue
prompt.format(topic="dogs")  # "Tell me a joke about dogs"
```

### Constructor

```python
@classmethod

    cls,
    messages: Sequence[MessageLike | tuple[str, str]]
) -> ChatPromptTemplate
```

Create a chat prompt template from a list of messages.

<ParamField path="messages" type="Sequence" required>
  List of message templates. Each can be:
  - A tuple like `("human", "Hello {name}")`
  - A `BaseMessage` instance
  - A `BaseMessagePromptTemplate`
  - A `MessagesPlaceholder`
</ParamField>

### Example

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history"),
    ("human", "{input}")
])

result = prompt.invoke({
    "history": [("human", "Hi"), ("ai", "Hello!")],
    "input": "What's the weather?"
})
```

## FewShotPromptTemplate

Template for few-shot learning with examples.

**Source:** `langchain_core.prompts.few_shot`

**Inherits:** `BasePromptTemplate`

### Properties

<ResponseField name="examples" type="list[dict] | None">
  List of example dictionaries
</ResponseField>

<ResponseField name="example_selector" type="BaseExampleSelector | None">
  Selector to choose examples dynamically
</ResponseField>

<ResponseField name="example_prompt" type="PromptTemplate" required>
  Template for formatting each example
</ResponseField>

<ResponseField name="suffix" type="str" required>
  Suffix template appended after examples
</ResponseField>

<ResponseField name="prefix" type="str" default="''">
  Prefix template prepended before examples
</ResponseField>

<ResponseField name="example_separator" type="str" default="'\\n\\n'">
  String separator between examples
</ResponseField>

### Example

```python
from langchain_core.prompts import FewShotPromptTemplate, PromptTemplate

example_prompt = PromptTemplate.from_template(
    "Question: {question}\nAnswer: {answer}"
)

prompt = FewShotPromptTemplate(
    examples=[
        {"question": "What's 2+2?", "answer": "4"},
        {"question": "What's 3+3?", "answer": "6"}
    ],
    example_prompt=example_prompt,
    suffix="Question: {input}\nAnswer:",
    input_variables=["input"]
)

prompt.format(input="What's 4+4?")
```

## ImagePromptTemplate

Template for creating prompts with images.

**Source:** `langchain_core.prompts.image`

**Inherits:** `BasePromptTemplate`

### Properties

<ResponseField name="template" type="str | dict" required>
  Image URL template or image data template
</ResponseField>

### Example

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.prompts.image import ImagePromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("human", [
        {"type": "text", "text": "What's in this image?"},
        {"type": "image", "source": {"type": "url", "url": "{image_url}"}}
    ])
])

prompt.invoke({"image_url": "https://example.com/image.jpg"})
```
